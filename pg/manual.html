Manual

Se você já usou o Word do seu dispositivo eletrônico, então as funções do Word não são novidade para você, entretanto caso não tenha conhecimento este manual irá te ajudar a conhecer melhor as funções do site.

Lembrando que este modelo não é uma cópia perfeita do Word, e sim um bloco de notas com funcionalidades a mais.

- Este botão é usado para deixar as palavras selecionadas dentro da caixa de texto em negrito. Ex: Negrito

- Este botão é usado para deixar as palavras selecionadas dentro da caixa de texto em itálico. Ex: Itálico

- Este botão é usado para deixar as palavras selecionadas dentro da caixa de texto sublinhadas. Ex: Sublinhado

- Este botão é usado para deixar as palavras selecionadas com um risco no centro. Ex: Riscadas

- A expressão "número elevado" pode ser usada com este botão para elevar um número ou letra. Ex: 48¹

- Quando precisar digitar uma letra ou número abaixo da linha de texto em tamanho reduzido, use este botão

- Quando precisar criar um link em um trecho, pode usar esse botão para digitar o link e linkar ele a palavra.

- Este botão faz o contrário do anterior, ao invés de linkar, ele quebra a ligação do link com o texto acoplado.

- Quando precisar organizar o texto, com este botão poderá colocar o texto da direita para esquerda.

- Este botão coloca o texto no centro da linha, alinhando eles com um espaço pra direita e esquerda.

- Esse faz a função contrária do primeiro botão de organizar o texto, ele segue da esquerda para direita com o texto.

- Com este botão você pode organizar o texto de forma justificada, deixando espaçamentos iguais entre as palavras.

- Quando uma redação deixamos um espaço a cada início de parágrafo, este botão cria esse espaço.

- O botão anterior cria o espaço do inicio do texto, este pode ser usado pra fazer o contrário, no caso retrair a frase.

- Os H são usados como tamanhos pré configurados para aumentar ou diminuir titulos, subtitulos e texto.

- Caso tenha digitado errado alguma informação no texto, você pode usar este botão para reverter o erro.

- Caso use o botão anterior sem querer e necessite voltar para oque estava antes, pode usar esse botão para reverter.

- Nesta caixa começa do número 1 ao 7, é usado para aumentar ou diminuir o tamanho das letras e números.
